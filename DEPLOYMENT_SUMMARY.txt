HOW TO DEPLOY (CHOOSE ONE METHOD)
================================================================================

METHOD 1: AUTOMATED SCRIPT (EASIEST)
-------------------------------------
Windows:
  1. Open Command Prompt in project folder
  2. Run: deploy.bat
  3. Follow prompts
  4. Go to vercel.com/new and import your repo
  5. Click Deploy

Linux/Mac:
  1. Open Terminal in project folder
  2. Run: chmod +x deploy.sh
  3. Run: ./deploy.sh
  4. Follow prompts
  5. Go to vercel.com/new and import your repo
  6. Click Deploy

METHOD 2: MANUAL DEPLOYMENT
----------------------------
1. Initialize Git:
   git init
   git add .
   git commit -m "Initial commit - RPIN application"

2. Create GitHub repo at: https://github.com/new
   Name: rpin
   Make it Public

3. Push to GitHub:
   git remote add origin https://github.com/YOUR_USERNAME/rpin.git
   git branch -M main
   git push -u origin main

4. Deploy on Vercel:
   - Go to: https://vercel.com/new
   - Sign in with GitHub
   - Import your rpin repository
   - Click Deploy
   - Wait 2-3 minutes
   - Done!

METHOD 3: VERCEL CLI
--------------------
1. Install: npm install -g vercel
2. Login: vercel login
3. Deploy: vercel --prod

================================================================================
TROUBLESHOOTING
================================================================================

Issue: "Git not found"
Fix: Install Git from https://git-scm.com/downloads

Issue: "Can't push to GitHub"
Fix: Make sure you created the repo first
     Check your GitHub username in the URL

Issue: "Vercel deployment fails"
Fix: Check build logs in Vercel dashboard
     Verify all files are committed to Git

Issue: "Frontend can't connect to API"
Fix: Check browser console (F12)
     Frontend auto-detects API URL
     Should work automatically on Vercel

Issue: "Module not found"
Fix: Ensure api/index.py exists
     Check that backend/ folder structure is intact
