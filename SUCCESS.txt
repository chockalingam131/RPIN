HOW TO TEST THE FRONTEND
================================================================================

Option 1: Use the Test Page (EASIEST)
--------------------------------------
1. Open the file: test-api.html (in your project folder)
2. Update the API URL to your Vercel URL
3. Fill in the form:
   - Village: Theni
   - Crop: Tomato
   - Quantity: 1000 kg
   - Date: (pick a future date)
4. Click "Test Prediction API"
5. See the results!

Option 2: Use curl (Command Line)
----------------------------------
curl -X POST https://your-app.vercel.app/api/v1/predict \
  -H "Content-Type: application/json" \
  -d '{
    "village_location": "theni",
    "crop_type": "tomato",
    "quantity_kg": 1000,
    "harvest_date": "2024-03-15"
  }'

Option 3: Use Browser Console
------------------------------
1. Open: https://your-app.vercel.app/
2. Press F12 (Developer Tools)
3. Go to Console tab
4. Paste this code:

fetch('https://your-app.vercel.app/api/v1/predict', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    village_location: 'theni',
    crop_type: 'tomato',
    quantity_kg: 1000,
    harvest_date: '2024-03-15'
  })
})
.then(r => r.json())
.then(data => console.log(data));
